# mmt-donut-chart
MMT-DONUT-CHART with Chart.js.

##Options
```javascript
var options = {
	animationStep: 100,
	chartColor: '#0bbaba',
	cutOut: 98,
	bgColor: '#444444',
	doughnutPadding: 5,
	dotColor: '#0bbaba',
	outerCanvasSize: 250,
	pointSize: 5
};
```
